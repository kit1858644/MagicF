            ---------- Welcome to use MagicF! ----------

簡介:
    此軟件可幫助你隱藏電腦上的文件(普通搜索是搜不出來的)。

按鍵說明:
    1.主按鍵:  如有已隱藏文件/文件夾, 展開管理(把文件/文件夾拖放到主按鍵, 即可隱藏文件)
    2.init:    初始化程式
    3.Unhide:  恢復已隱藏文件/文件夾
    4.Ex:      提取已隱藏文件夾內的文件/文件夾(提取物會自動取消隱藏)
    5.Path:    選擇提取物儲存的路徑
    6.Extract: 提取文件
    7.<--:     反回展開管理

    * 右鍵 - About: 軟件有關信息

注意:
    1.最新版本可在T1me中找到
    2.此軟件只支持64bit系統(已測試在Windows 7 ,8 ,8.1 ,10)
